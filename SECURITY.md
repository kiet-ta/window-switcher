# Security Policy

We treat the security of our Wayland IPC layer and system operations with extreme priority. 

## Supported Versions

Only the **latest stable release** of the Hyprland Visual Window Switcher receives active security backports. 

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you discover a potential vulnerability within the window switcher (such as arbitrary code execution via forged Hyprland client titles or unsafe `tmpfs` traversal), please **do not** open a public issue.

Instead, please email the maintainer directly at **trananhkiet21082005@gmail.com**. 

We will acknowledge receipt of your vulnerability report within **48 hours**, and we strive to send you regular updates about our progress. If the vulnerability is accepted, we will publish a patch alongside a comprehensive CVE advisory thanking you for your discovery.
